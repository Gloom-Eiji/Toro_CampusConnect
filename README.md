# Toro_CampusConnect
Links directed to Toro Campus Connect project repositories; frontend, backend, and mobile deployment.

[Campus Connect Backend Repo](https://github.com/CSUN-Campus-Connect/campus-connect-server)

[Campus Connect Froendend (Web App) Repo](https://github.com/CSUN-Campus-Connect/campus-connect-client)

[Campus Connect Mobile Deployment Repo](https://github.com/CSUN-Campus-Connect/campus-connect-expo)

This project is all about creating a unified digital hub for the CSUN community. We're consolidating scattered tools into one mobile-first experience to enhance campus life and foster a trusted environment. This sprint marked a pivotal shift for us, moving from foundational infrastructure to active feature completion. We've expanded our Events API, integrated AWS S3 for media, and are now preparing for our mobile beta. Let's dive into the specific goals that guided our work over the last two weeks.
